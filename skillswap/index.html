Introduction

Abstract

SkillSwap (Real-Time Resource Pool) is an innovative web-based platform designed to democratize education through peer-to-peer skill exchange. The platform implements a unique time-based economy where users earn credits by teaching and spend them to learn, eliminating financial barriers to quality education.

Built using modern full-stack technologies including React 19, Node.js, Express.js, and MySQL, the application features secure JWT authentication, real-time messaging, user profiles with skill portfolios, and an intuitive responsive interface.

This project demonstrates the viability of community-driven learning platforms and provides a foundation for future enhancements including video calling, AI-based skill matching, and mobile applications.

Conclusion

SkillSwap serves as a fully functional proof-of-concept for a demonetized, community-driven education platform. It successfully implements full-stack web development (React + Node.js + MySQL) with secure JWT authentication, real-time features (chat, online status), and modern responsive UI design.

The platform demonstrates the viability of peer-to-peer learning through time banking, providing a foundation for future expansion into a production-ready application.
